FlowJo SweetSpot Web

Load a FlowJo-exported .csv or .xlsx, tune the panel parameters, and run the sweet-spot analysis directly in the browser.

Inputs

Population base is the parent gate used in the FlowJo export, for example CHO/Singlets.

Detector max is the upper instrument limit used to estimate saturation risk.

UpperEnvelope = Median_pos + k * rSD_pos

Minimum acceptable SI is the global separation threshold used for all titratable dyes before the app starts preferring lower spill.

Dye configuration lets you edit only the fluorescence channel and the color used in the plots.

By default, GFP is treated as informational_only: it still contributes spill and can still be invaded by other dyes, but it is not treated as a titration to optimize.

Metric trends

Stain Index shows separation from the unstained control. Higher is better.

Ws is the weighted spill / spread penalty into the other channels. Lower is better.

Quality Score is a descriptive combined score, but the final decision does not rely only on it.

Upper envelope shows detector pressure using Median + k * rSD. Staying lower is safer.

Color is assigned by dye and symbol by clone, and that same mapping is reused across figures. You can edit dye colors in the configuration table.

Selection criteria

Each dye gets its own decision-score panel.

Y axis: Decision score, so higher means a better compromise between enough SI and low spill.

X axis: concentration.

Black outlined point: the selected condition.

The score stops rewarding extra brightness strongly once SI is already sufficient, so a brighter point can lose if its spill penalty is much larger.
