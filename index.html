FlowJo SweetSpot Web

Load a FlowJo-exported .csv or .xlsx, tune the panel parameters, and run the sweet-spot analysis directly in the browser.

Inputs

Population base es el gate padre en el export de FlowJo, por ejemplo CHO/Singlets.

Detector max es el límite superior del instrumento para estimar riesgo de saturación.

Minimum acceptable SI es el piso de separación mínima antes de evaluar invasión. Concentraciones con SI por debajo de este valor se descartan primero.

Dye configuration permite editar el canal detector y el color en los plots.

GFP está marcado como informational_only: contribuye a la invasión en otros canales y puede ser invadido, pero no se optimiza su concentración.

Metric trends

Stain Index = (Mediana_teñido − Mediana_ST) / (2 × rSD_ST). Mide cuánta señal hay sobre el ruido del control sin tinción. Más alto es mejor.

Invasion (Ws) = suma de (fracción_de_células × SI_aparente_en_canal_vecino) para cada canal vecino. Mide cuánto se ve el colorante como señal real en los otros canales. Más bajo es mejor.

Score = SI / (1 + Invasion). El pico de esta curva indica la concentración óptima: suficiente señal sin arruinar los canales vecinos.

Color por colorante, símbolo por clon. Se pueden editar los colores en la tabla de configuración.

Selection criteria

Cada colorante tiene su propio panel de Score.

Eje Y: Score = SI / (1 + Invasion). El pico de esta curva es la concentración óptima.

Eje X: concentración.

Punto con borde negro: concentración seleccionada.

Una concentración más alta puede tener mayor SI pero también mayor invasión, haciendo que el Score caiga. El pico captura ese balance.
